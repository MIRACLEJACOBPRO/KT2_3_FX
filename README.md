# KT2_3_FX

风险路径识别模块 - 行为链路分析与风险评估系统

## 项目说明

本项目实现了基于行为链路的风险路径识别、风险评分计算和风险报告生成功能。

## 主要功能

1. 行为链路构建模块 (behavior_chain_builder.py)
2. 风险路径识别模块 (FX.py)
3. 风险评分计算
4. 风险报告生成

## 项目结构

- FX.py - 风险路径识别主模块
- behavior_chain_builder.py - 行为链路构建模块
- fx_data.py - 数据处理模块
- output/ - 输出目录
  - chains/ - 行为链路文件
  - risk_report.txt - 风险报告
- all_data/ - 数据目录
